Se requiere de la biblioteca Psutil, para ello abra una terminal y ejecute:
	pip install psutil

Ejecute el main que le pedirá cuál análisis hacer:

Análisis 1:
	Determina el tiempo y uso porcentual en memoria y CPU de cada algoritmo

Análisis 2:
	Determina el consumo en Watts de cada algoritmo y cuanto tardan en ejecutarse
	
	Para este análisis se necesita del comando powermetrics que está disponible para 	MacOS
